**Приложение библиотеки**


Информационная система обслуживания библиотеки содержит следующую информацию:

+ названия книг
+ ФИО авторов
+ наименования издательств
+ год издания
+ количество страниц
+ количество иллюстраций
+ стоимость
+ название филиала библиотеки


Информационная система обслуживания библиотеки позволяет:

1. для указанного филиала посчитать количество экземпляров указанной книги, находящихся в нем
2. для указанной книги посчитать количество факультетов, на которых она используется \
в данном филиале, и вывести названия этих факультетов
3. добавить и изменить информацию о книгах в библиотеке
4. добавить и изменить информацию о филиалах
5. поддерживать срабатывание хуков на пользовательские исключительные ситуации
